Node Read File

The Node 'readFile' function is a built-in method of the Node program Object object().

Node Read File Form

The 'Node Read File' form allows the end-user to enter the URI of a text or binary file.

There is a toggle at the right border of the input box that occludes the 'r' on the word number.

The toggle will allow the end-user to scroll through a list of whole numbers when deciding which numeral to square.

Once entered, the end-user clicks the checkbox to affirm his or her selection and then fires off an event by clicking the submit button.

The button is wired to the node 'readFile' function via the node 'fs' module'. the node 'readFile' function will read the file located at the URI selected and return via an alert the result of the read.

Optionally, a separate 2nd button may fire off the binary of a second selection, as well.
