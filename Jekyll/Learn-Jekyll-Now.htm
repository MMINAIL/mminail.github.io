Learn Jekyll

Welcome! This is the Learn Jekyll page of the MMINAIL.

Note. Jekyll is best served as a Ruby gem. Therefore, you must have an updated copy of Ruby on your machine.

How To Start The Local Jekyll Server In Mac Os Sierra

  1. Go to your locally configured repo in the Finder
  2. Right click over the folder name
  3. Expose the Services menu
  4. Open a 'new Terminal at folder'

Once a Terminal window is open with the directory pointing to the folder of your repo ...

  1. Input the following line of code at the Terminal $ prompt
Starting the local Jekyll server ...
{% highlight html %}
jekyll serve --watch --baseurl ""
{% endhighlight %}
Server now running... press control-c to stop.

Note. Once you have entered the local Jekyll server start command into the Terminal, you may subsequently simply scroll up the Terminal history to accept the command in the future.

  1. Now, open up your chrome browser
  2. And, type this address in the URL location bar of your browser window ... http://localhost:4000/

  3. Hit the Enter key to navigate to the local rendering of your repo
