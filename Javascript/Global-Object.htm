Global Object

The global object in the Javascript language, if rendered in a browser, is the browser window.

Therefore, when accessing variables and objects placed under the global object simply use the keyword this , as follows: 


Given: The following global var named 'a' ...


{% highlight html %}
var a = "Hello There!";
// From the Terminal prompt $
a
// Returns
"Hello There!"
// From the Terminal prompt $
this.a
// Returns
"Hello There!"
// From the Terminal prompt $
window.a
// Returns
"Hello There!"
{% endhighlight %}


¡Recuerde! window = this

If we substitute an object, such as a function ( instead of a global var ), the same outcome still applies, as follows: 


Given: The following function named 'b' ...


{% highlight html %}
function b() {
  "use strict";
}
// From the Terminal prompt $
b
// Returns
function b() {
  "use strict";
}
// From the Terminal prompt $
this.b
// Returns
function b() {
  "use strict";
}
// From the Terminal prompt $
window.b
// Returns
function b() {
  "use strict";
}
{% endhighlight %}


Remember! this = window
