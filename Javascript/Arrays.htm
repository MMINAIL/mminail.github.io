Onclick

What if the end-user has his or her javascript turned 'off' at the browser?

Will your hyperlinks still 'work'?

Some websites flat-out tell the surfer they require Javascript to be turned on for full fuctionality. See: The Economist Magazine.

So, for external links, this author uses the Javascript method shown below.

Note. This method is now deprecated in favor of unobtrusive javascript.

Unobtrusive javascript keeps the javascript separate from the html and css. 

For external links ...

{% highlight javascript linenos %}
// Setting the target page via Javascript
onclick="javaScript:this.target='_blank';"
{% endhighlight %}

Perfecto! ... 

Internally, this author sets the target page using the old reliable Html4 way, as follows: 

For internal links ...

{% highlight html linenos %}
// Setting the target page via html
target="_self"
{% endhighlight %}

Que guapo! ...

Intro to a Simple Javascript function ...

Instructions
  1. Open up developer tools from your Chrome browser
  2. Seek out the Javascript console
  3. Enter the following script ( copy and paste )
Javascript Call The Square of X
{% highlight javascript linenos %} // Javascript Call The Square of X function square(x) { return x * x; }; {% endhighlight %}

Now, hit the enter key ...

The Javascript console will return a value of 'undefined'.

  • Next, "call" the function of the script
  • Be sure to include an entry for the now declared variable 'x'
{% highlight javascript linenos %} //Here, we'll 'square' the numeral seven (7) square(7); //Hit enter {% endhighlight %}

Javascript will return the sum of the products ( square ) of 'x'.

{% highlight javascript linenos %} //Javascript returns 49 //The 'chevron' prompt returns > {% endhighlight %}

Once the script is entered into the Javascript console ... The code is retained by the browser in a type of short-term memory ( cache ) until the current session ends.

So, feel free to try another number!

{% highlight javascript linenos %} //Here, we'll 'square' the numeral nine (9) square(9); //Javascript will return ... //the sum of the products ( square ) of 'x' 81 {% endhighlight %}
