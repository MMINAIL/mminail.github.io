Do While and While Loops Topic

The key question of this article is ... "When do we use a While Loop?" versus "When to use a 'Do While' Loop?"

When To Use a While Loop

If you only wish to iterate ... if and only if the condition set is true, then use a 'While Loop'.

For, if the condition is false in a 'While Loop', an iteration will never be performed.

When To Use a Do-While Loop

If on the other hand, you wish to run the iteration at least once regardless of whether or not the condition set is true or false, then use a 'Do-While' Loop.

In a 'Do-While' Loop at least one single iteration will automatically takes place PRIOR to any check on the condition set.

So, if you wish to run the iteration at least once, use a 'Do-While' Loop.

If you never want the iteration mechanism to run if the condition set is not true, then use a 'While Loop'.

Otherwise, feel free to use a 'Do-While Loop'.
