Classes in Javascript

Welcome! This is the Classes in Javascript page of the MMINAIL.

To create a class in Javascript, first name it and open up a set of right and left curly brackets {...} to accept your code. 

Set the name of the class ...
{% highlight javascript linenos %}
class Donor {

}
{% endhighlight %}
Get your Firefox Web-console ready ...

Class Constructors

Next, we set up the Class constructor. The Class constructor is established with the 'constructor' keyword and the opening up of another set of right and left curly brackets {...} to accept your code.

The 'camelCase' names of the properties of the newly named class are fed to the contructor as arguments. The arguments are then enveloped within a set of left and right parethesis (...), as follows: 

Set the class constructor ...
{% highlight javascript linenos %}
class Donor {
    constructor(firstName, lastName, eMailHandle, 
    eMailDomain, eMotiCon) {
    //Assign all properties to the 'this' variable
    }
}
{% endhighlight %}
Get your Firefox Web-console ready ...

Class Properties

Now that we have effectively declared and named the Class properties in advance as arguments to the constructor function, it is time to assign each property to its respective 'this' variable, as follows:' 

Assign the class properties ...
{% highlight javascript linenos %}
class Donor {
    constructor(firstName, lastName, eMailHandle, 
    eMailDomain, eMotiCon) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.eMailHandle = eMailHandle;
        this.eMailDomain = eMailDomain;
        this.eMotiCon = eMotiCon;
    }
}
{% endhighlight %}
Get your Firefox Web-console ready ...

Class Methods

New methods can now be added that will manipulate the properties of the Constructor arguments in back-ticks. 

Set the class methods ...
{% highlight javascript linenos %}
class Donor {
    constructor(firstName, lastName, eMailHandle, 
    eMailDomain, eMotiCon) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.eMailHandle = eMailHandle;
        this.eMailDomain = eMailDomain;
        this.eMotiCon = eMotiCon;
    }

    displayback() {
        return `The console will now reveal a message 
        via the alert box ...

        Given. The key to a proper donor record is 
        the capture of a valid electronic mail address.
        
        Dialoque

        A valid email address is comprised of a valid 
        handle, such as '${this.eMailHandle}' and a 
        valid domain, such as '${this.eMailDomain}'.
        
        Though the importance of the capture of the 
        donor's first name, for example ... 
        '${this.firstName}' and the capture of the 
        donor's last name, for example ... 
        '${this.lastName}' is readily apparant ...
        
        The concatenation of the donor's email address 
        including the email handle '${this.eMailHandle}' 
        plus the 'at' character, or "@" ...
        
        Plus, the domain name '${this.eMailDomain}' 
        is of critical importance to the donation puzzle.
        
        How else to broadcast the monthly newsletter 
        without a set of valid, opt-in email addresses?

        ${this.eMotiCon}
        
        The Management`;
    }
}
{% endhighlight %}
Get your Firefox Web-console ready ...

Instantiation

Set up a variable outside the Class function to pass new values to the arguments of the constructor function.

Thus, in effect instantiating an instance of the new class. 

Instantiate a new instance ... Given.
Constructor(firstName, lastName, eMailHandle,
eMailDomain, eMotiCon)
{% highlight javascript linenos %}
var author = new Donor("Adam", "Webber", 
    "cannabuds", "live.com", 
    "\u{1F600}");
{% endhighlight %}
Get your Firefox Web-console ready ...

Execution

Finally, set up a command statement outside the Class function to execute the 'displayback' method of the 'Donor' class. 

Execute the 'displayback' method of the 'Donor' class ...
{% highlight javascript linenos %}
window.alert(author.displayback());
{% endhighlight %}
