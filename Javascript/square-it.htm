Square It! ( in Javascript )

Welcome! This is the Square It! ( in Javascript ) page of the Carbon Free Footprint project.

Displaying the Program

The following code block represents the current version of the subject program written in Javascript by the author(s) at the Rwebaz repos.

Note. Carriage returns ' ➡️ ' have been added to the code lines when needed to faciliate viewing in a landscape mode on smaller viewports such as smartphones and tablets. 

The program as originally written in Javascript, as is ...
{% highlight javascript %}
//Square It
"use strict";
//Declare and assign the arrow function to accept one argument
let SQ = (nbr) => {
  //Set the input variable locally within the function
  var nbr = prompt("Please enter an integer ...")
  /* where nbr = any whole integer */
  //Assign the calculation to a local variable
  var sq = nbr * nbr;
  /* where sq = square of the number */
  //Alert the result to the end-user
  window.alert("The square of the number " ➡️ 
  + nbr + " given is " + sq);
  //Return and archive the result at the console
  return("The square of the input " + nbr + " = " + sq);
};
{% endhighlight %}

Rendering the Result

Ok ... it's time to give this program a whirl!

Open up a new window in the Firefox browser and click on the 'open menu' aka the 'three-bars icon' to the far right of the now unpopulated 'search or enter an address' bar.

Next, select the category 'Developer'.

Finally, select the Web Developer tools submenu by clicking 'Web Console'.

You will now wish to go ahead and copy and paste the above code into the program entry line at the bottom of the console.

Note. You can also alternatively simply copy and paste the code from the original (.js) program file, as well. 

When entered, the program initially returns ...
{% highlight javascript %}
￩ "use strict"
{% endhighlight %}

Not to worry ... We are using EcmaScript 6 'strict mode', so thank you to the Javascript engine for rendering that "use strict" text in the console window!

By now, it is the time to type the command needed into the Web Console entry line ( at the bottom of the console ) in order to execute our function SQ in the console window. 

Execute the function SQ with an open argument ...
{% highlight javascript %}
//The execution command for the SQ function is ...
SQ();
/* Notice the open argument accepts the prompt nbr */
{% endhighlight %}

Upon excution, a prompt-box pops up to show the end-user where to enter his or her subject squarable digit(s) ie.) the real number that will be squared by the end-user within the function.

Upon enter, secondly ... an alert-box pops up to show the end-user what the result of the calculation is!

Finally, to end the function, the result of the calculation is returned to the console as an archive of the activity. 

The archive of the activity; where input = 100
{% highlight javascript %}
￩ "square of the input = 10000"
{% endhighlight %}
