Canvas Subtitle Blocks

The following subtitle uses the <canvas> element plus a simple javascript function to render a scaleable block of "made-on-the-fly" text.

SVG Pictures

After the rendering of the subtitle ( bordered in red ), a Scaleable Vector Graphic (SVG) image is displayed using an embedded <img> tag within a <picture> element.

The result is the following rendition of the Arizona Cannabis Dispensary Logo Badge tested for responsiveness on a virtual Galaxy S5 smartphone emulator via Google's Chrome Developer Tools.
