How To Convert a (.png) To an (.svg) via ( AI ) Adobe Illustrator

Adobe Illustrator CS6 has a "save as" feature whereby a fully loaded (.png) can be translated directly into a compressed (.svgz) file.

However, I prefer to first load the (.png) or other lossy or lossless photo into GIMP and then change the size of the image to suit a desktop screen rendering.

Next, I export the file from GIMP as as an Adobe Photoshop (.psd) file.

From there, I can now open the (.psd) in Adobe Illustrator (AI) by converting all layers to objects, if any, and then invoking the Adobe Illustrator CS6 transparency grid to ensure no background noise has been rendered during the import.

Finally, after selecting "save as" (.svg), a browser-viewable image-text file renders that I can now point to regular way whenever a standard image is called for.

For example, the following result is a rendition of the Medical Marijuana Initiative of North America - International Limited USPTO Registered Logo Badge tested for responsiveness on a virtual Galaxy S5 smartphone emulator via Google's Chrome Developer Tools.
