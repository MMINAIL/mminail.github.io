Visual Studio Code for Mac - User Snippets

Welcome! This is the Visual Studio Code - User Snippets page of the Carbon Free Footprint project.

To engage the Visual Studio Code editor for the purpose of editing the internal User Snippets (.json) page on your MacBook Pro, simply ...

  1. Click the Code tab in the upper left hand corner of the top menu border.
  2. The Preferences tab will appear ... Hover over the Preferences tab.
  3. Click User Snippets.
  4. Scroll down to select your code language ( in this case ... HTML )

Code, Preferences, User Snippets

Now that we have the User Snippets (.json.) file engaged, its time to go to work!

Optional. Use $1, $2 for tab stops, ${id} and ${id:label} and ${1:label} for variables.

Note. Variables with the same id are connected.

Notice the (.json) page block opens { with a set of curly braces that also closes } at the tail-end of the code block ... In between go each individual user snippet, as follows:

Note. Be sure to place a comma , after each user block snippet.

Exception. The last record before the closing } curly brace does not require a comma ,.

If you place a comma , after the last record ... An error warning in the form of underlined red-text will be generated by the code editor.

Note. The "body" of each snippet record is enclosed in double " quotes. Therefore, if your snippet has a bit of code that requires enveloping its elements in quotes ... Such as an tag, the please remember to use single ' quotes when enveloping the elements. 

Remember, inside double quotes, single quotes must be used ...
{% highlight html %}
{
"Print to console example": {
		"prefix": "log",
		"body": [
			"console.log('$1');",
			"$2"
		],
		"description": "Log output to console"
	},

"Basic Hyperlink": {
		"prefix": "hyp-basic",
		"body": [
			""
		],
		"description": "Create Basic Hyperlink"
	},

"Bitly Hyperlink": {
		"prefix": "hyp-bit",
		"body": [
			"Bitly hyperlink service."
		],
		"description": "Create Bitly Hyperlink"
	}
}
{% endhighlight %}
