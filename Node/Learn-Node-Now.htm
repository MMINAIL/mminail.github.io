Learn Node

Welcome! This is the Learn Node page of the MMINAIL.

Node is a program that understands how to use Javascript outside of the browser ... in the Terminal program of your Mac.

Installation

You can easily download both node and the npm package manager for node libraries in one (1) pre-packaged Binary Installer Package, or BIP from Nodejs dot org.

Note. The installation of node defaults to the /usr/local/bin/ subdirectory of your development machine.

That is where the BIP will install the node executable.

Also, the installation of npm defaults to the /usr/local/bin/ subdirectory of your development machine, as well.

That is where the BIP will install the npm executable.

Be sure to run an 'echo' from your Terminal prompt in order to ensure the $PATH variable indeed has an export for the default /usr/local/bin/ subdirectory location.

Otherwise, when the executables are tapped, you'll get nothing from your install but an error message in your Terminal that reads ... 

Check your User $PATH variable!
{% highlight html linenos %}
-bash: node: command not found
{% endhighlight %}

To confirm install of npm, from the Terminal prompt $ ...

{% endhighlight %}
Shows the now operable version of npm.
