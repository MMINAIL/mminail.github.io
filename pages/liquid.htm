Site dot time

The site dot time pipe date and time liquid statement accepts a set of short parameters, as follows:

Note. The double curly brace wrapper \{\{ ... \}\} has been removed from the code statement to prevent execution. 

Highlighting Liquid code w Rouge via Liquid ...

{% highlight liquid linenos %}
site.time | date: "Date: %a, %b %d, %Y
Time: %X %Z"
{% endhighlight %}
