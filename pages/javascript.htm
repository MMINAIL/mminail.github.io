Onclick

What if the end-user has his or her javascript turned 'off' at the browser? Will the hyperlinks still 'work'? For external links, this author uses the javascript method shown below. Internally, this author sets the target page using the reliable old html way, as follows: 

For external links ...

{% highlight javascript linenos %}
// Setting the target page via Javascript
onclick="javaScript:this.target='_blank';"
{% endhighlight %}
For interal links ...

{% highlight html linenos %}
// Setting the target page via html
target="_self"
{% endhighlight %}
