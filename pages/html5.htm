Html Pre and Code

Intro to Html Pre and Code tags ...

Use triple backticks and a code language qualifier to wordwrap and highlight lines of css from within a pre and code combo block.

This solution ( triple backticks and a code language qualifier ) also renders a code block when commenting both before and after the review of a pull request at GitHub, both locally via GitHub for desktop, and at the GitHub remote repository. 

Highlighting Css code w Rouge using triple backticks and a code language qualifier.

{% highlight css linenos %}
```css

pre {
    white-space: pre-wrap;
    /* white-space: pre; */
    background-color: rgb(255,255,229);
	/* light yellow */
    background: hsl(30,80%,90%);
	/* full moon yellow */
    display: block;
    font-family: monospace;
    margin: 1em 0;
}

code { 
    background: hsl(220, 80%, 90%);
	/* sky blue */
}

```
{% endhighlight %}
