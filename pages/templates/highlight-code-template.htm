Css dot largo

Establish the 'largo' class via css by setting the double-xx font size for the Immediate tag, as follows: 

Setting the double-xx font size for the Immediate tag
{% highlight css linenos %}
i.largo {
    font-size: xx-large;
}
{% endhighlight %}
