Now, that works fine. But, how will the code execute in Strict Mode? One way to find out is to transfer our code to a (.js) page and invoke Stict Mode at the top of the program. 

Invoke Strict Mode ...

'use strict';
{% endhighlight %}
Place at the top of program.
