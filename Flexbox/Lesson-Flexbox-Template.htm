Flex Box Container

Welcome! This is the Flex Box page of the Carbon Free Footprint project.

The Flex Box Container allows content and elements to respond to changing viewports.

Flex Box Test

This is a test of the Flex Box for the responsiveness of a pre block tagged with the Css class "flex-box".

Note. When configuring the flex box along the main axis ( as opposed to the 'cross axis') ...

The flex items ( elements ) of the flex container will line up horizontally, one after the other in perfect succession across the main axis ( horizontal ) of the flex container when the flex-direction is switched to 'row' in the css.

Alternatively, the flex items ( elements ) of the flex container will line up vertically, one after the other in perfect succession down the cross axis ( vertical ) of the flex container when the flex-direction is switched to 'column' in the css, as follows: 

		Testing the Flex Box Container in Column mode
		{% highlight css linenos %}
		/* Wordwrap and highlight the lines of code  */
		pre {
			display: block;
			font-family: Arial, Helvetica, sans-serif;
			font-size: small;
			white-space: pre-wrap;
			margin: 1em 0;
			background-color: rgb(255,255,229); /* light yellow */
		}
		/* Configure the flex box along the cross axis */
		pre.flex-box {
				display: flex;
				flex-direction: column;
		}
		{% endhighlight %}
		Discarding other Flex Box Container options
		{% highlight css linenos %}
		/* Discards: The parent pre container */
		pre {
			/* display: inline; */
			/* font-family: monospace; */
			/* white-space: pre; */
			/* height: 100%; */
			/* background: hsl(30,80%,90%); /* full moon yellow */
		}
		/* Discards: Configuring the flex box */
		pre.flex-box {
			/* flex-direction: row; where the default option is 'row' */
			/* flex-direction: row-reverse; */
			/* flex-direction: column-reverse; */
		
			/* -webkit-flex-flow: row wrap;
			-ms-flex-flow: row wrap;
			flex-flow: row wrap; */
		
			/* -webkit-flex-flow: row;
			-ms-flex-flow: row;
			flex-flow: row; */
		
			/* margin-right: -0.625rem;
			margin-bottom: 0.75rem;
			margin-left: -0.625rem; */
		}
		{% endhighlight %}
		Finito! ...
