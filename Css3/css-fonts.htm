How To Form A Font Face In Css3

  • Start w a media query ...
  • Add a font-famly ...
  • Fill in the src urls ...
{% highlight css linenos %} @font-face { font-family: 'fontawesome'; src: url('../fonts/glyph/fontawesome-webfont.eot'); src: url('../fonts/glyph/fontawesome-webfont.eot?#iefix') format('embedded-opentype'), url('../fonts/glyph/fontawesome-webfont.woff2') format('woff2'), url('../fonts/glyph/fontawesome-webfont.woff') format('woff'), url('../fonts/glyph/fontawesome-webfont.ttf') format('truetype'), url('../fonts/glyph/fontawesome-webfont.svg') format('svg'); font-weight: normal; font-style: normal; } {% endhighlight %}
  • Finish w a normal font-weight and font-style
