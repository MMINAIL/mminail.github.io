Ruby Version Manager

Ever wonder why you should use a RVM, or Ruby Version Manager?

Well, for starters ...

Not only does a RVM make installing multiple ruby interpreters / run-times easy and consistent...

The RVM also provides features such as 'gem-sets' that are not typically supported out of the box on most ruby installs.

And, as a bonus, RVM lets you run multiple different copies of rubies in separate terminal shells concurrently!
