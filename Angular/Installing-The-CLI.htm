How To Install Angular CLI In Mac Os Sierra

Welcome! This is the Installing the CLI page of the MMINAIL.

Note. This page has explicit access to the local jQuery library as well as a local complimentary working (.js) file.

Installing The CLI

To install the Angular Command Line Interface, or CLI onto your development machine, first install both node and the Node Package Manager, or NPM.

After the developer has installed both node and and the Node Package Manager, or NPM onto his or her development machine, the Angular CLI may then be installed using npm, as follows: 


From the Terminal prompt $ ...


{% highlight html %}
npm install -g @angular/cli
{% endhighlight %}

Updating The CLI

Hey! The Angular Team just came out with a new, updated version of the Command Line Interface, or CLI.

How do I upgrade the CLI?

To upgrade the CLI, you must first counter-intuitively uninstall the CLI and cleanse the cache!

As follows: 


From the Terminal prompt $ ...


{% highlight html %}
npm uninstall -g @angular/cli cache clean
{% endhighlight %}

Once the old version of the CLI has been removed from your development machine and the cache has been cleansed, you may simply next re-install the CLI, as above: 


From the Terminal prompt $ ...


{% highlight html %}
npm install -g @angular/cli
{% endhighlight %}
